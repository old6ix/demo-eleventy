# 11ty Demo

A demo for [11ty](https://www.11ty.dev/).

![Preview](https://user-images.githubusercontent.com/23132866/205863308-10593a2d-d271-4706-872c-ddaa2d18f262.png)

# Usage

## Install dependencies

```sh
npm install
```

## Generate site

```sh
npm run build
```

## Serve site

```sh
npm run serve
```
