# 11ty Demo

A demo for [11ty](https://www.11ty.dev/).

![Preview](https://user-images.githubusercontent.com/23132866/205854782-03516a3b-1f40-48fa-b9b0-fe9c72a60197.png)

# Usage

## Install dependencies

```sh
npm install
```

## Generate site

```sh
npm run build
```

## Serve site

```sh
npm run serve
```
